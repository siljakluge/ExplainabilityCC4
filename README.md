# Heuristic_CAGE_CHALLENGE_4
This repository contains a heuristic approach for solving the Cage Challenge 4. It includes agent implementations, evaluation scripts, and supporting utilities for running and analyzing heuristic-based solutions in the CAGE-4-MARL environment.

# Structure
Heuristic_CAGE_CHALLENGE_4: Main directory for heuristic agents and evaluation scripts.
CybORG/: Core environment and simulation code for the challenge.
documentation/: Project documentation and guides.

# Getting Started
Install dependencies:
'''bash
pip install -r requirements.txt
'''
Run the heuristic agent or evaluation scripts as needed.
'''bash
python3 evaluation.py --max-eps 20
'''
# Usage
Implement or modify agents in Heuristic_CAGE_CHALLENGE_4.
Use provided scripts to train, evaluate, or test agents.
Results and logs are saved in the Results/ directory.# ExplainabilityCC4
